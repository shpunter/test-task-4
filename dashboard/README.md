dashboard/src/UI - folder with shared/generic components\
dashboard/src/pages - folder with app pages


## Dashboard app

### install dependency
```bash 
deno install
```
### run app
```bash 
deno task dev
```

### run tests 
```bash 
deno test
```

## Render app
### install dependency
```bash 
npm i
```

### run app 
```bash 
npm run build && npm run serve 
# should be up while running dashboard app
```

### run tests
```bash 
npm run test
```
