## Render app

### install dependency

```bash
npm i
```

### run app

```bash
npm run build && npm run serve 
# should be up before running dashboard app
```

### run tests

```bash
npm run test
```

## Dashboard app

### install dependency

```bash
deno install
```

### run app

```bash
deno task dev
```

### run tests

```bash
deno test
```
